# 電子書管理後台

## 簡介
這是一個電子書管理後台，提供了以下功能：
- **CRUD 電子書**：新增、修改、刪除和查詢電子書。
- **使用者查詢**：查詢使用者資料。
- **管理員 CRUD**：管理後台使用者帳號（新增、修改、刪除）。
- **借閱記錄查詢**：查詢電子書的借閱記錄。
- **報表功能**：生成各類報表。

此專案使用 **React + Redux** 作為前端框架，**Node.js + PostgreSQL** 作為後端架構，並透過 **Apollo Client** 實現 GraphQL 的數據交互。

## **功能展示**
